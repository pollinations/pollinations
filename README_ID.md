## 🚀 Unified API Baru — Sudah Live

Kami baru saja merilis **<https://gen.pollinations.ai>** — satu Endpoint tunggal untuk semua kebutuhan generation AI kamu: teks, gambar, audio, video — semuanya jadi satu.

### Apa yang Baru

* **Endpoint terpadu** — satu API di `gen.pollinations.ai` untuk semua jenis generation
* **Kredit Pollen** — sistem pay-as-you-go yang simpel ($1 ≈ 1 Pollen)
* **Semua model, satu tempat** — Flux, GPT-5, Claude, Gemini, Seedream, dan masih banyak lagi
* **API Key** — publishable key untuk frontend, secret key untuk backend
* **CLI** — `npx @pollinations_ai/cli` untuk manusia maupun AI agent ([source](https://github.com/pollinations/pollinations/blob/main/packages/polli-cli))

> Mulai dari [enter.pollinations.ai](https://enter.pollinations.ai) dan cek [API docs](https://enter.pollinations.ai/api/docs)

### Generate Audio

**Endpoint GET simpel:**

```
curl 'https://gen.pollinations.ai/audio/Hello%20from%20Pollinations?voice=nova&key=YOUR_API_KEY' -o speech.mp3
```

**Kompatibel dengan OpenAI TTS:**

```
curl 'https://gen.pollinations.ai/v1/audio/speech' \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -d '{"model": "tts-1", "input": "Hello from Pollinations!", "voice": "nova"}' \
  -o speech.mp3
```

Voice yang tersedia: `alloy`, `echo`, `fable`, `onyx`, `nova`, `shimmer`, plus [30+ voice ElevenLabs](https://enter.pollinations.ai/api/docs).

### MCP Server untuk AI Assistant

MCP (Model Context Protocol) server kami memungkinkan AI assistant seperti Claude untuk generate gambar dan audio secara langsung. [Pelajari lebih lanjut](https://github.com/pollinations/pollinations/blob/main/packages/mcp/README.md)

#### Konfigurasi

Tambahkan ini ke konfigurasi MCP client kamu:

```
{
  "mcpServers": {
    "pollinations": {
      "command": "npx",
      "args": ["@pollinations_ai/mcp"]
    }
  }
}
```

### Jalankan dengan npx (tanpa instalasi)

```
npx @pollinations_ai/mcp
```

Alternatif dari komunitas seperti [MCPollinations](https://github.com/pinkpixel-dev/MCPollinations) dan [Sequa MCP Server](https://mcp.sequa.ai/v1/pollinations/contribute) juga tersedia.

AI assistant bisa:

* Generate gambar dari deskripsi teks
* Membuat audio text-to-speech dengan berbagai pilihan voice
* Memutar response audio lewat speaker sistem
* Mengakses semua model dan layanan pollinations.ai
* Menampilkan daftar model, voice, dan kapabilitas yang tersedia

Untuk penggunaan yang lebih advanced, cek [dokumentasi API](https://github.com/pollinations/pollinations/blob/main/APIDOCS.md) kami.

## 🔐 Autentikasi

Ambil API Key kamu di [enter.pollinations.ai](https://enter.pollinations.ai)

### Jenis Key

| Key | Prefix | Use Case | Rate Limit | Status |
| --- | --- | --- | --- | --- |
| **Publishable** | `pk_` | Client-side, demo, prototype | 1 pollen per IP per jam | ⚠️ Beta |
| **Secret** | `sk_` | Khusus server-side | Tanpa rate limit | Stable |

> ⚠️ **Publishable key:** Proteksi Turnstile akan segera hadir. Belum direkomendasikan untuk production.

> ⚠️ **Jangan pernah expose key `sk_`** di kode client-side, git repo, atau URL publik

> 💡 **Sedang bikin aplikasi?** Pakai [Bring Your Own Pollen](https://github.com/pollinations/pollinations/blob/main/BRING_YOUR_OWN_POLLEN.md) — user bayar sendiri untuk pemakaian mereka, kamu bayar $0

### Pembatasan Model

Setiap API Key bisa dibatasi ke model tertentu. Saat membuat key di [enter.pollinations.ai](https://enter.pollinations.ai), kamu bisa:

* **Izinkan semua model** — key bisa dipakai untuk model apapun yang tersedia
* **Batasi ke model tertentu** — pilih model mana saja yang boleh diakses key tersebut (misalnya, hanya `flux` dan `openai`, atau cuma `gptimage-large`)

### Cara Pakai

```
curl 'https://gen.pollinations.ai/image/a%20cat?key=YOUR_KEY'
```

**Environment variable (best practice):**

```
export POLLINATIONS_API_KEY=sk_...
```

### Image Generation

import requests

def download_image(prompt):
    url = f"https://pollinations.ai/p/{prompt}"

    with open('generated_image.jpg', 'wb') as file:

## 🗂️ Struktur Proyek

Codebase kami dibagi jadi beberapa folder utama, masing-masing punya peran spesifik dalam ekosistem pollinations.ai:

* [`pollinations.ai/`](https://github.com/pollinations/pollinations/blob/main/app): Aplikasi React utama untuk website Pollinations.ai.
* [`image.pollinations.ai/`](https://github.com/pollinations/pollinations/blob/main/image.pollinations.ai): Backend service untuk image generation dan caching pakai Cloudflare Workers dan R2 storage.
* [`text.pollinations.ai/`](https://github.com/pollinations/pollinations/blob/main/text.pollinations.ai): Backend service untuk text generation.
* [`packages/polli-cli/`](https://github.com/pollinations/pollinations/blob/main/packages/polli-cli): CLI Pollinations — untuk manusia, AI agent, dan segala sesuatu di antaranya.
* [`packages/sdk/`](https://github.com/pollinations/pollinations/blob/main/packages/sdk): Library SDK NPM dengan fungsi-fungsi siap pakai untuk Pollinations.ai.
* [`packages/ompc/`](https://github.com/pollinations/pollinations/blob/main/packages/ompc): Oh My Polli Code, mekanisme Open Code Router siap pakai yang dibangun di atas framework open code tapi dengan model-model Pollinations.
* [`packages/mcp/`](https://github.com/pollinations/pollinations/blob/main/packages/mcp): Model Context Protocol (MCP) server untuk AI assistant seperti Claude agar bisa generate gambar secara langsung.

Struktur ini mencakup frontend website, backend service untuk image dan text generation, serta integrasi seperti Discord bot dan MCP server, memberikan framework yang komprehensif untuk platform pollinations.ai.

Untuk setup development dan pengelolaan environment, lihat [DEVELOP.md](https://github.com/pollinations/pollinations/blob/main/DEVELOP.md).

## 🤖 Contoh Praktis: Chatbot Pembuat Gambar Interaktif (Node.js)

Selain lewat `curl` atau browser, kamu juga bisa langsung ngobrol dengan API Pollinations lewat terminal, layaknya chatbot pribadi buat generate gambar. Skrip Node.js di bawah ini bikin sesi interaktif sederhana: kamu ketik prompt, skrip mengirim request ke endpoint image generation, lalu hasilnya otomatis disimpan sebagai file `.png` di folder yang sama.

Yang menarik, setiap file yang di-generate diberi nama unik berdasarkan `timestamp` saat itu juga (`art-<timestamp>.png`). Jadi kamu nggak perlu khawatir file lama ke-overwrite setiap kali generate gambar baru — cocok banget kalau kamu lagi iterasi banyak prompt sekaligus.

**Cara pakai:**

1. Install dependency `axios` dulu: `npm install axios`
2. Ganti `YOUR_API_KEY_HERE` dengan API Key Pollinations kamu
3. Jalankan skripnya: `node chatbot.js`
4. Ketik prompt gambar apa saja, tekan Enter, tunggu proses selesai, lalu ulangi. Ketik `keluar` kapan saja untuk mengakhiri sesi.

```javascript
const axios = require('axios');
const fs = require('fs');
const path = require('path');
const readline = require('readline');

// Ganti dengan API Key Pollinations Anda
const API_KEY = "YOUR_API_KEY_HERE";
const MODEL = "flux";

// Membuat antarmuka input di terminal
const rl = readline.createInterface({
    input: process.stdin,
    output: process.stdout
});

async function generateImage(promptText) {
    // Membuat nama file otomatis berdasarkan waktu agar tidak saling menimpa
    const timestamp = Date.now();
    const fileName = `art-${timestamp}.png`;
    const outputPath = path.join(__dirname, fileName);
    const url = `https://gen.pollinations.ai/image/${encodeURIComponent(promptText)}?model=${MODEL}&width=1024&height=1024&nologo=true`;

    console.log(`\n⏳ Sedang memproses: "${promptText}"...`);

    try {
        const response = await axios({
            method: 'get',
            url: url,
            headers: {
                'Authorization': `Bearer ${API_KEY}`
            },
            responseType: 'stream'
        });

        const writer = fs.createWriteStream(outputPath);
        response.data.pipe(writer);

        return new Promise((resolve) => {
            writer.on('finish', () => {
                console.log(`✅ BERHASIL! Gambar disimpan sebagai: ${fileName}\n`);
                resolve();
            });
            writer.on('error', (err) => {
                console.error("❌ Gagal menulis file:", err);
                resolve();
            });
        });
    } catch (error) {
        console.error("❌ Gagal generate gambar:", error.message);
    }
}

// Fungsi utama chatbot terminal
function startChatbot() {
    rl.question('💬 Masukkan prompt gambar Anda (atau ketik "keluar" untuk berhenti): ', async (input) => {
        if (input.toLowerCase() === 'keluar') {
            console.log('Sampai jumpa!');
            rl.close();
            process.exit(0);
        }
        if (!input.trim()) {
            console.log('Prompt tidak boleh kosong!');
            startChatbot();
            return;
        }

        await generateImage(input);

        // Tanya kembali setelah selesai
        startChatbot();
    });
}

console.log("=== POLLINATIONS CHATBOT GENERATOR ===");
startChatbot();
```

> 💡 **Catatan:** Contoh ini memakai endpoint `gen.pollinations.ai/image/` dengan Query Parameter `model`, `width`, `height`, dan `nologo`. Kalau kamu mau pakai model lain, tinggal ganti value dari variabel `MODEL` di bagian atas skrip.
