Run the setup script:
curl -fsSL https://raw.githubusercontent.com/pollinations/pollinations/main/apps/hermes/setup-pollinations.sh \ | bash -s -- YOUR_API_KEY
  • Saves API key to ~/.hermes/config.yaml
  • Adds 7 models: Kimi K2.5, DeepSeek, GLM, Gemini + Search, Claude Haiku + 2 paid
  • Sets Kimi K2.5 as default with DeepSeek + GLM fallbacks
Add this to ~/.hermes/config.yaml:
model: provider: custom default: kimi base_url: https://gen.pollinations.ai/v1 api_key: YOUR_API_KEY context_length: 256000 custom_providers: - name: pollinations base_url: https://gen.pollinations.ai/v1 api_key: YOUR_API_KEY api_mode: chat_completions models: kimi: context_length: 256000 deepseek: context_length: 128000 glm: context_length: 128000 gemini-search: context_length: 128000 claude-fast: context_length: 200000 claude-large: context_length: 200000 gemini-large: context_length: 1000000 fallback_model: provider: custom model: deepseek base_url: https://gen.pollinations.ai/v1 api_key: YOUR_API_KEY mcp_servers: pollinations: command: "npx" args: ["-y", "@pollinations_ai/model-context-protocol"]
Then set model: hermes model custom:pollinations:kimi

Image, video & audio via MCP

Hermes supports MCP natively. Add the pollinations.ai MCP server for multi-modal generation.

hermes mcp add pollinations -- npx -y @pollinations_ai/model-context-protocol

This gives your agent image generation (Flux, GPT Image, Imagen 4), video (Veo 3.1, Seedance), text-to-speech (13 voices), and vision analysis — all through Pollinations.
