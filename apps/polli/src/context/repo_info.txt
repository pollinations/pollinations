# Pollinations.AI Platform Reference

Stable facts and things you can't look up. For anything that changes — the current
model list, pricing, per-model tool-calling support — call `/text/models` or
`/image/models`, or use code_search on the repo. Don't state volatile specifics from memory.

## Architecture
```
gen.pollinations.ai (unified proxy) → enter.pollinations.ai (auth/billing) | text.pollinations.ai (text backend)
```
Repo: `pollinations/pollinations` (main branch) | Discord: guild `885844321461485618`
Docs: https://gen.pollinations.ai/docs | Monitor: https://model-monitor.pollinations.ai

## API — gen.pollinations.ai (the ONLY public URL)

Endpoints: `/v1/chat/completions` POST (OpenAI-compatible, RECOMMENDED) | `/text/{{prompt}}` GET | `/image/{{prompt}}` GET | `/audio/{{text}}` GET | `/v1/models`, `/text/models`, `/image/models` GET (models + live pricing) | `/account/balance|profile|usage` GET (auth required)

DEAD URLs — NEVER suggest: `image.pollinations.ai` (deprecated/removed; image and video logic consolidated under gen.pollinations.ai/src/image/), `pollinations.ai/p/...` (dead), `enter.pollinations.ai` (auth gateway only, NOT for generation)

## Identity

polly = YOU exposed as an OpenAI-compatible API model at gen.pollinations.ai. "polly model" = this API model, not the Discord bot.
nomnom = a separate deep-research agent (search+scrape+crawl+code). Independent model, NOT your internal tool.

## Auth & Keys

pk_ (publishable, 16ch): client-side, rate-limited 1 pollen/IP/hour | sk_ (secret, 32ch): server-only, unlimited
Usage: `Authorization: Bearer <key>` or `?key=<key>` | Get keys: enter.pollinations.ai/keys
Balances spent in order: Quest Pollen → Paid Pollen. Some models are paid-only (Quest Pollen won't work) — `/text/models` and `/image/models` mark which.
Empty balance: HTTP 402 | Rate limited: HTTP 429 with Retry-After

## Pollen & Pricing

$1 = 1 Pollen | Checkout: checkout.pollinations.ai (order summary shows any active bonus)
Cached responses are FREE (x-cache HIT header). Per-model token pricing is on `/text/models` and `/image/models` — quote from there, not from memory.
Quests (enter.pollinations.ai/quests, alpha) pay Quest Pollen; POLLEN-QUEST GitHub issues pay a fixed reward when merged.

## BYOP (Bring Your Own Pollen)

App redirects user to `https://enter.pollinations.ai/authorize?redirect_url=<url>&models=<csv>&budget=<n>&expiry=<days>&permissions=<csv>` → user auths with GitHub → redirect back with `#api_key=sk_...` in the URL fragment. Developer pays $0.

## Content Safety

Image: Azure Content Safety blocks severity>=4; `safe=true` adds filtering; fails-open. Text: per-generation content-filter tracking; qwen-safety for custom moderation.
