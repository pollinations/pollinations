## API - gen.pollinations.ai

**Base URL**: `https://gen.pollinations.ai`
**Docs**: https://enter.pollinations.ai/api/docs/open-api/generate-schema
**Model Status**: https://model-monitor.pollinations.ai

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/v1/models` | GET | All available models |
| `/text/models` | GET | Text-only models |
| `/image/models` | GET | Image-only models |
| `/text/{prompt}` | GET | Simple text generation |
| `/image/{prompt}` | GET | Simple image generation |
| `/v1/chat/completions` | POST | OpenAI-compatible chat (recommended) |

**Rate Limits**:
- `sk_` (secret key): Unlimited - for server-side use
- `pk_` (publishable key): Rate limited by tier

  - Seed: 3/hr (show GitHub activity to upgrade)
  - Flower: 10/hr
  - Nectar: 20/hr
