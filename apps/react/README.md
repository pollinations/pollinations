# react.pollinations.ai

Public showcase for Pollinations React UI surfaces.

The public app is organized around:

- `Models` - model families, modality selectors, and SDK examples.
- `Primitives` - curated UI building blocks for product workflows.
- `Compositions` - auth, wallet, model picker, and prompt composer patterns.

The exhaustive primitive coverage page is still available for internal visual
QA at `?view=showcase`, but it is intentionally not linked from the public UI.

## Development

```bash
npm run dev
npm run typecheck
```

Run from `apps/react`.
