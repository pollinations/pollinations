Draw a classic L-system fractal plant as pure ASCII art and save it to output.txt in the current directory.

Rules:
- axiom: X
- X -> F+[[X]-X]-F[-FX]+X
- F -> FF
- "+" turns right, "-" turns left, "[" pushes state, "]" pops state
- iterate 4-5 times

Render it as ASCII onto a 2D character grid using Python 3 stdlib only. Use "#" or "*" for stems, "." or " " for empty cells. Target roughly an 80x40 grid. Save the final art to output.txt, then cat output.txt so you can see it.

If the first render looks wrong (off-grid, too sparse, too dense), adjust parameters (angle, step size, iterations, grid size) and try again.
