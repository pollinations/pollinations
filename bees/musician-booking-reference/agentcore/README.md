# AgentCore runtime

Amazon Bedrock AgentCore's HTTP contract expects:

- ARM64 container
- host `0.0.0.0`
- port `8080`
- `POST /invocations`
- `GET /ping`

This reference bee implements that contract in `src/providers/aws/agentcore.ts`
and serves it with:

```bash
npm run serve:agentcore
```

The `agentcore.example.json` file is intentionally illustrative because
AgentCore CLI project files are generated by `agentcore create`. Use the
container build mode and point it at `Dockerfile.agentcore`.
