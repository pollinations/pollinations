# Pollinations Pitch Deck — Style Prompt

Generate a 8-slide seed round pitch deck from the uploaded source content. One point per slide. Follow this design system.

## Style
**Cozy 8-bit pixel art.** Stardew Valley meets hackerspace. Warm, playful, builder-culture. Never corporate, never cyberpunk, never dark.

## Colors (Strict)
- Background: Warm cream `#FFFDF5` with soft peach-to-mint horizontal gradient
- Text: Near-black `#111111`
- Accent: Lime green `#ecf874` — use for highlighted numbers, key labels, stat callout backgrounds
- Supporting: Soft pastels (lavender, sky blue, pale pink)
- Never pure white, never dark backgrounds, never neon

## Typography
- Headlines: Bold pixel font (Press Start 2P), ultra-large, 30-50% of slide area
- Body: Clean monospace (Space Mono), small
- Key numbers: Massive bold monospace with lime-green highlight behind them
- Jump ratio: Headlines to body at minimum 8:1

## Characters (rotate across slides)
- **Polly the Bee** — 16-bit cute bee mascot, sparkle trails
- **Nomnom** — round brown mochi creature, represents creators
- **Monitor Robot** — retro CRT TV with face/arms/legs, represents platform
- **Creator Boy** — brown-haired kid at pixel computer, green shirt

## Visual Motifs
- Dashed pixel borders on every slide (game UI frame), small squares in corners
- Circuit boards growing from plants, code brackets `{ }` as stems
- Pixel coins, money bags, isometric floating islands
- Lime-green sparkle particles around key numbers
- Footer: `Pollinations.AI — Seed Deck` bottom-left

## Layout Rules
- 1 slide = 1 message, never overload
- Stat slides: grid of rounded cards, massive numbers, tiny labels
- Problem/solution: split layout (illustration left, text right)
- Flowcharts: circular pixel arrows with character icons at each node
- Competitive: isometric 3-layer stack (gray bottom, blue middle, lime-green top)
- Roadmap: 3-layer building — bottom done, middle under construction, top blueprinted
- Characters interact with diagrams (holding coins, pointing at charts, sitting on layers)

## Tone
Confident, genuine, technically literate. No buzzwords. "We built", "creators earn", "here's how it works."
